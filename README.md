## Install
1. ```git clone https://github.com/hossein-taheri/Music-Player-Vue.js```
1. ```cd Music-Player-Vue.js```
1. ```npm install```
1. Start the app in development mode via ```quasar dev```
1. Build the app for production via ```quasar build```
